CommandDescriptionSample Code
dlg.get_combobox_sel("name")Get the position of the item selected in the combo boxComboBox = ["1", "2", "3", "4", "5"] When "4" is selected Input: a = dlg.get_combobox_sel("ComboBox") Output: > a = 3
