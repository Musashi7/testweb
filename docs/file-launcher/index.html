File Launcher

Interface#

ButonDescription
SRegister the Python file
CClear the registered Python file
RReplace the registered Python file with another Python file
FuncExecute the registered Python program

Operating procedure#

  1. Click the S button and select the Python file

  2. When you select a file, the green mark to the left of the Function button turns red

  3. Click the Function button to execute the program
