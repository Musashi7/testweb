Editor

Generate surface mesh to selected parts.

Description#

This command generates surface mesh to selected part . It will be used with {ref}Meshing.SetMeshAttribute command. This function support triangular and quadrilateral element.

Syntax#

Meshing.SurfaceMeshing(crlPart, surfaceMesh, bUseSetting=True, bFMesher=False,
iThreadNum=8, bRefData=True, bMeshColor=False, iPartColor=65280)

Macro: {ref}Macro-Meshing-SurfaceMeshing2D

Ribbon: {menuselection}Meshing --> Surface Meshing

Inputs#

crlPart : A Cursor List specifying the parts needed to create mesh. : This is a required input.

surfaceMesh : A {ref}SURFACE_MESH specifying the mesh parameter. : This is a required input.

bUseSetting : A Boolean to enable or disable the use of local setting. If this parameter is disabled (False), the existing local mesh setting will not be used. : The default value is True.

bFMesher : A Boolean to enable or disable the use of F-mesher. F-mesher is a mesher to create only Quad4 elements. If surfaceMesh is set to create Tri3 element, this options does not have any effect. : The default value is False.

iThreadNum : An Integer specifying the number of CPU threads for parallel meshing. This mesher create parallel meshing for each independent (no assembled face) part on each thread. If 2 parts are assembled together, they will be treated as a single part for parallel meshing. This parameter must be smaller than the maximum number of CPU cores. : The default value is 8.

bRefData : A Boolean to enable or disable the creating of Reference data. Reference data is used to store CAD model. If this parameter is enabled (True), Reference data will be kept after meshing. : The default value is True.

bMeshColor : A Boolean to enable or disable the use of mesh color. If this parameter is enabled (True), the result meshed parts will be colored as described by iPartColor. : The default value is False.

iPartColor : An Integer specifying the color of result meshed parts. : The default value is 65280 (white).

Return Code#

A String of 1 if success, or 0 if fail.

Sample Code#

Geometry.Part.Cube()
Meshing.SetMeshAttribute(crlPart=[Part(1)], surfaceMesh=SURFACE_MESH
(dGeomAngle=0.7853981634, iOptLevel=5, dAutoMergeTinyFacesAngle=0.5235987756,
bGeomApprox=True, iNextEntityOffsetId=0))
Meshing.SurfaceMeshing(crlPart=[Part(1)], surfaceMesh=SURFACE_MESH(dGeomAngle=0.7853981634,
iOptLevel=5, dAutoMergeTinyFacesAngle=0.5235987756, bGeomApprox=True, iNextEntityOffsetId=0))
