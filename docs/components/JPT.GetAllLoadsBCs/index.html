JPT.GetAllLoadsBCs()

Description#

Get all the information of all existing loads and boundary conditions.

Syntax#

JPT.GetAllLoadsBCs()

Inputs#

This utility function does not require any input value.

Return Code#

DItemVector object containing a list of existing loads and boundary conditions with their information.

Sample Code#

# Prepare simple sample
Geometry.Part.Cube(iColorPart=6409934)
Geometry.Part.Cube(dlOrigin=[0.012, 0.0, 0.0], strPartName="Cube_2", iColorPart=13259210)
Geometry.Part.Cube(dlOrigin=[0.024, 0.0, 0.0], strPartName="Cube_3", iColorPart=7697908)
BoundaryConditions.FixedConstraint(crlTarget=[Face(76)])
BoundaryConditions.Pressure.General(crlTarget=[Face(52)])
# Get the information of all existing loads and boundary conditions
loadsNBCs = JPT.GetAllLoadsBCs()
JPT.Debugger(loadsNBCs)
